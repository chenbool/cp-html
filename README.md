# cp-html 网页模板

HTML5 网页模板，包含首页和登录页。

## 项目结构

```
cp-html/
├── assets/                  # 静态资源
│   ├── css/                # 样式文件
│   │   ├── bootstrap.css  # Bootstrap 样式
│   │   ├── common.css     # 公共样式
│   │   ├── font-awesome.css # Font Awesome 图标
│   │   ├── iconfont.css  # 字体图标
│   │   ├── index.css     # 首页样式
│   │   └── login.css     # 登录页样式
│   ├── font/              # 字体文件
│   ├── fonts/             # Font Awesome 字体
│   ├── img/               # 图片资源
│   │   ├── header/       # 头部图片
│   │   ├── index/        # 首页图片
│   │   ├── left/         # 侧边栏图片
│   │   ├── login/        # 登录页图片
│   │   └── right/        # 右侧图片
│   └── js/               # JavaScript 文件
├── index.html              # 首页
├── login.html              # 登录页
└── README.md
```

## 页面说明

| 页面 | 说明 |
|------|------|
| `index.html` | 网站首页 |
| `login.html` | 用户登录页 |

## 技术特点

- 响应式布局，适配多种设备
- 使用 Bootstrap 框架
- Font Awesome 图标字体
- 简洁的 UI 设计

## 依赖

- Bootstrap
- Font Awesome
- jQuery

## 使用方法

直接用浏览器打开 HTML 文件即可预览：

```bash
# 打开首页
open index.html

# 打开登录页
open login.html
```

## 浏览器支持

- Chrome
- Firefox
- Safari
- Edge
- Internet Explorer 10+
